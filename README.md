# Tosania Blog - 功能更新报告

首先感谢原模板作者与开源社区提供的优秀基础：本项目基于 Fuwari/Astro 生态进行二次开发与定制。

## 本次完成内容

### 1) 首页与路由结构调整
- 默认首页改为 `Home`（`/`）。
- 原文章流首页迁移为 `Blog`（`/blog/`）。
- 导航调整为：`Home / Blog / Archive / About / poems / GitHub`。

### 2) Blog 分类体系（四类）
新增文章分区概念：
- `research`（科研类）
- `tech`（技术类）
- `life`（生活类）
- `hidden`（隐藏类）

说明：
- 支持在文章 frontmatter 中通过 `section` 显式指定分区。
- 未指定时，会根据标题/标签/分类关键词自动推断到公开分区（科研/技术/生活）。
- `hidden` 默认不进入公开 Blog 流与公开统计。

### 3) Blog 分类选择入口
- 在导航栏 `Blog` 位置提供下拉分类入口（全部/科研类/技术类/生活类）。
- 移动端菜单同步支持分类子项。

### 4) 隐藏模式（彩蛋）
新增全站隐藏模式，触发命令：`yyc`。

隐藏模式开启后：
- 全站切换为隐藏主题视觉（暗色氛围）。
- 自动替换背景图与侧栏头像。
- Blog 入口自动导向隐藏列表（`/blog/hidden/`）。
- 只显示 `section: hidden` 的文章。

隐藏模式关闭后：
- 自动恢复正常 Blog 流与普通主题访问路径。
- 退出体验已优化（包含页面回退与状态同步）。

### 5) 隐藏内容访问链路
- 隐藏列表页：`/blog/hidden/`
- 隐藏详情页：`/blog/hidden/[...slug]/`
- 支持多层级 slug 的隐藏文章访问。
- 隐藏列表卡片样式已与普通 Blog 卡片保持一致（仅色系随隐藏模式变化）。

### 6) 界面细节优化
- 移除顶部明暗切换按钮（按当前需求）。
- 隐藏模式下关闭 Blog 分类选择（桌面/移动端均禁用子项选择）。
- 隐藏模式下侧栏无关模块收敛，减少干扰。

## 使用说明（当前版本）

### 标记隐藏文章
在文章 frontmatter 中添加：

```yaml
section: hidden
```

### 触发隐藏模式
在页面非输入框区域直接键入：

```text
yyc
```

再次输入 `yyc` 可退出隐藏模式。
